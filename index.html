About The Event

Join us for an engaging workshop: 'Build a Machine Learning Model using Azure Custom Vision.' In this hands-on session, participants will delve into Azure's powerful capabilities, exploring how to create and deploy custom image recognition models. Perfect for beginners and intermediates interested in AI and machine learning, this workshop promises an insightful journey into the world of Azure Custom Vision. Mark your calendars and come ready to learn, experiment, and take the first steps toward mastering machine learning with Azure.
